## üöÄ Getting Started

1. Clone the project
  ```bash
  git clone https://github.com/digita-webshop/digita-webshop-backend.git
  ```
2. Install project dependencies
  ```bash
  npm install
  #or
  yarn add
  ```
 
3. Run the development server:

  ```bash
  npm run dev
  # or
  yarn dev
  ```
